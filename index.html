Сайт Компании
Разработка сайта компании под ключ. Веб-сайт для компании - это прежде всего информационное представительство компании в сети. Разработка данного сайта включает множество технических аспектов.
Процесс Разработки
1. Согласование технического задания
На данном этапе мы с Вами решим, какого типа сайт нужен, каким образом сайт будет выглядеть, что сайт будет уметь делать. Помогу составить текст для сайта. Если уже имеется текст для будущего сайта, то я проверю его на оригинальность. От того, насколько уникален текст сайта, зависит его место в поисковой системе. Если уже есть макет и сформировано техническое задание, то, при необходимости, могу прокомментировать или прояснить какие-то технические нюансы. На данном этапе можно точно определить, какая у сайта будет цена, исходя из сложности технического задания. Для согласования технического задания могу приехать в любое удобное для Вас место в Санкт-Петербурге.
2. Оформление домена
Как только возникнет определённость в техническом задании, можно сразу оформить доменное имя.
3. Создание макета
Будет создан макет, который отвечает требованиям технического задания. Далее данный макет я отправлю Вам на проверку: именно таким Вы хотели бы видеть будущий сайт? Если Вас полностью устроит составленный макет, то можно приступить к следующему этапу. В случае, если не устроит, макет будет дорабатываться, пока у Вас не останется к нему замечаний.
4. Вёрстка
Этап написания кода интерфейса для будущего сайта. Используется HTML и CSS.
5. Фронтенд
Написание кода для анимации или оптимизации работы интерфейсов сайта. Для данной задачи используется JavaScript и вспомогательные библиотеки, такие, как React и JQuery.
6. Бэкенд
Если на сайте присутствует возможность оставлять комментарии, отправлять заявки, сообщения, регистрироваться и т.д., то необходимо составить программный код для серверной обработки этих процессов. Для этой задачи я использую язык PHP и фреймворк Symfony.
7. Загрузка сайта на хостинг
Как только сайт будет свёрстан и отлажен(пройдёт процедуру тестирования на какие-то программные ошибки), можно будет выложить его в открытый доступ. Для этого необходимо обратиться к компании, которая предоставляет услугу хостинга. Весь процесс размещения сайта на хостинге, а также регистрации его в Яндекс Вебмастер и Google Analytics я беру на себя. Это входит в стоимость сайта. Также в стоимость входит установка SSL-сертификата, настройка файла robots.txt и иконки сайта.
О проекте
Идея данного проекта заключается в том, чтобы создавать сайты с уникальным дизайном, которые становятся в поисковой системе выше прочих, за счёт чистого и валидного кода, оптимизации картинок, отсутствия тяжелых плагинов для анимации, быстродействия и т.д.
